int num_scales_per_octave ( ) const
inline

The smoothed image \(I_{2\sigma}\) is equivalent to the downsampled image \(I_{\sigma}\). Because of this observation, a pyramid of gaussians is divided into octaves.

We call an octave a stack of \(S\) blurred images \(I_{\sigma_i}\) separated by a constant factor \(k\) in the scale space, i.e., \( \sigma_{i+1} = k \sigma_i \). The number of scales in each octave is the integer \(S\) such that \(k^S \sigma = 2 \sigma\), i.e., \( k= 2^{1/S} \).

double scale_camera ( ) const
inline

Let us consider an image \(I\). In the scale-space framework, we work with the scale-space image function \( I : (x,y,\sigma) \mapsto (g_\sigma * I)\ (x,y) \), where \(g_\sigma\) is the gaussian distribution with zero mean and standard deviation \(\sigma\).

We also denote by \( I_\sigma : (x,y) \mapsto I(x,y,\sigma) \) the image \(I\) at scale \(\sigma\). Note that the real image is \(I_0: (x,y) \mapsto I(x,y)\). Because of the spatial sampling, the camera captures the real image \(I_0\) with some blurring and the camera-acquired image is \(I_{\sigma_\textrm{camera}}\).

Here cameraSigma() thus corresponds to \(\sigma_\textrm{camera}\).

double scale_geometric_factor ( ) const
inline

The sequence \( (\sigma_i)_{1 \leq i \leq N} \) follows a geometric progression, i.e., \(\sigma_i = k^i \sigma_0\).

Laplacians of Gaussians \( \nabla^2 I_{\sigma} \) can be approximated efficiently with differences of Gaussians \( I_{k\sigma} - I_{\sigma} \) for each \( \sigma = \sigma_i \)*without needing to renormalize*.

Indeed:

\begin{eqnarray*} \frac{\partial I_\sigma}{\partial \sigma} &=& \sigma \nabla^2 I_\sigma \\ I_{k\sigma} - I_{\sigma} &\approx& (k-1) \sigma^2 \nabla^2 I_\sigma \end{eqnarray*}

Here scale_geometric_factor() corresponds to the value \( k \).

The image pyramid discretizes image function \((x,y,\sigma) \mapsto I(x,y,\sigma)\) in the scale-space. With the discretization, the image pyramid will consist of a stack of blurred images \((I_{\sigma_i})_{1 \leq i \leq N}\).

Here initSigma() corresponds to \(\sigma_0\).
