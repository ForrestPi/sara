Test based on Harris-Stephens' idea on corner detection. Basically the Hessian matrix \(\mathbf{H}\) is estimated by finite differentiation.

\((x,y)\) is on an edge if the Hessian matrix \(\mathbf{H}\) satisfies the following criterion \( \frac{\mathrm{det}(\mathbf{H})}{\mathrm{tr}(\mathbf{H})} > \frac{(r+1)^2}{r} \), where \(r\) is the ratio between the eigenvalues of \(\mathbf{H}\) and corresponds to the variable edgeRatio.

Extremum position refinement in scale-space based on Newton's method. (cf. [Lowe, IJCV 2004] and [Brown and Lowe, BMVC 2002]).

Parameters
[in]Ithe input gaussian pyramid
[in]xintegral x-coordinate of the extrema
[in]yintegral y-coordinate of the extrema
[in]sscale index of the extrema
[in]ooctave index of the extrema
[in]imgPaddingSzThis variable indicates the minimum border size of the image. DoG extrema that ends being located the border are not refined anymore.
[in]numIterThis variable controls the number of iterations to refine the localization of DoG extrema in scale-space. The refinement process is based on the function DO::refineExtremum().

Let \(D : \mathbf{R}^3 \mapsto \mathbf{R}\) be the difference of gaussian function and \((x,y,\sigma)\) be the approximate position of a local extremum of \(D\).

If \((x,y,\sigma)\) is the current guess of the local extremum, the refinement procedure seeks to minimize the following objective function iteratively:

\begin{eqnarray*} \mathrm{minimize}_{\mathbf{h}} D(\mathbf{x}) + D'(\mathbf{x})^T \mathbf{h} + 1/2 \mathbf{h}^T D''(\mathbf{x}) \mathbf{h}^T . \end{eqnarray*}

In practice the gradient vector \(D'(\mathbf{x})\) and hessian matrix \(D''(\mathbf{x})\) are approximated by finite difference and one must check that the hessian matrix \(D''(\mathbf{x})\) is indeed positive-definite.

Likewise, if \(\mathbf{x}\) is a minimum, then one must check that \(D''(\mathbf{x})\) is negative-definite.

Otherwise, we cannot refine the position of the extremum.
