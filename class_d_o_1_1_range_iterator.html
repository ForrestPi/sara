Detailed Description

template<bool IsConst, typename T, int N, int StorageOrder = ColMajor>
class DO::RangeIterator< IsConst, T, N, StorageOrder >

N-dimensional iterator class. In any case the 'RangeIterator' class is a heavy object. It is mostly useful for differential calculus. Otherwise prefer using other iterator instead.
