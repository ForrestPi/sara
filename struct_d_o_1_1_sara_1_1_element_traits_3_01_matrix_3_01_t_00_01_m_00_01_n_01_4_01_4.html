Detailed Description

template<typename T, int M, int N>
struct DO::Sara::ElementTraits< Matrix< T, M, N > >

The specialized element traits class when the entry is a matrix. Again the matrix is viewed as a scalar. Additions and subtractions between matrices are OK but multiplication will be the point-wise matrix multiplication.

This may be questionable and this may change in the future.
