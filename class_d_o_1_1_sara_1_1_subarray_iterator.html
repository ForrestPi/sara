template<bool IsConst, typename T, int N, int StorageOrder>
class DO::Sara::SubarrayIterator< IsConst, T, N, StorageOrder >

Multidimensional iterator base class. The 'SubarrayIterator' class is a heavy object. It is mostly useful for differential calculus. When possible, prefer using more elementary iterator instead.
