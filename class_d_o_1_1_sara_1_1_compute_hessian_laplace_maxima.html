Localizes Hessian-Laplace maxima for a given image.

This function does the following:

  1. Constructs a gaussian pyramid \(\nabla g_{\sigma(s,o)} * I\) from the image \(I\), where \((s,o)\) are integers. Here: \(\sigma(s,o) = 2^{s/S + o}\) where \(S\) is the number of scales per octaves.
  2. Localize maxima in determinant of Hessians functions in each scale \(\sigma(s,o)\).
Parameters
[in,out]scale_octave_pairsa pointer to vector of scale and octave index pairs \((s_i,o_i)\). This index pair corresponds to the determinant of Hessians.
Returns
set of Hessian-Laplace maxima in std::vector<OERegion> in each scale-normalized determinant of Hessians.
