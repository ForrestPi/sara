[in]extremumThresthe response threshold which the DoG extremum absolute value \( \left| \left( g_{\sigma(s+1,o)} - g_{\sigma(s,o)} \right) * I \right| (\mathbf{x}) \) must exceed. Note that \( \sigma(s',o') = 2^{s'/S+o'}\) where \(S\) is the number of scales per octave.
[in]edgeRatioThresthe Hessian matrix \(\mathbf{H}\) at the local scale-space extremum must satisfy

\[ \frac{\mathrm{det}(\mathbf{H})}{\mathrm{tr}(\mathbf{H}} > \frac{(r+1)^2}{r} \]

where \(r\) corresponds to the variable edgeRatioThres. In terms of implementation, we use the function DO::onEdge(). We use the \(r=10\) as stated in [Lowe, IJCV 2004].

Localizes DoG extrema for a given image.

This function does the following:

  1. Constructs a gaussian pyramid \(\nabla g_{\sigma(s,o)} * I\) from the image \(I\), where \((s,o)\) are integers. Here: \(\sigma(s,o) = 2^{s/S + o}\) where \(S\) is the number of scales per octaves.
  1. Localize extrema in each difference of Gaussians \(\left( g_{\sigma(s+1,o)} - g_{\sigma(s,o)} \right) * I \) where \((s,o)\) are scale and octave indices.
Parameters
[in,out]scaleOctavePairsa pointer to vector of scale and octave index pairs \((s_i,o_i)\). This index pair corresponds to the difference of Gaussians \(\left( g_{\sigma(s_i+1,o_i)} - g_{\sigma(s_i,o_i)} \right) * I\) where the extremum \((x_i,y_i,\sigma_i)\) is detected.
Returns
set of DoG extrema in std::vector<OERegion> in each difference of Gaussians \(\left( g_{\sigma(s+1,o)} - g_{\sigma(s,o)} \right) * I \).
Image<float> DO::scaleAdaptedHarrisCornerness ( const Image< float > &  I,

Computes the Harris-Stephens corner function \(det(\mathbf{A}) - \kappa trace(\mathbf{A})^2\) where \(\mathbf{A}\) is the auto-correlation matrix. Good values of \(\kappa\) range in \([0.04, 0.15]\).

As a reminder, the auto-correlation matrix \(A\) is defined as below. Let \(g_{\sigma}\) be the Gaussian kernel with standard deviation \(\sigma\). And define the convolved image by \(I_\sigma = g_{\sigma_I} * I\), then

\[ \mathbf{A} = g_{\sigma_D} * \left[ \nabla I_{\sigma_I} \nabla I_{\sigma_I}^T \right] \]

.
