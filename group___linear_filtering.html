void DO::convolveArray ( T *  signal,
const typename ColorTraits< T >::ChannelType *  kernel,
int  signalSz,
int  kernelSz 
)

"Convolves" a 1D signal \(f\) (or 1D array), with a kernel \(g\).

Parameters
[in,out]signalthe 1D array containing the 1D signal \( f = (f_i)_{1\leq i \leq N}\), the resulting signal \(f*g\) is stored in signal.
[in]kernelthe "convolution" kernel \(g = (g_i)_{1 \leq i \leq K}\).
[in]signalSzthe signal size \(N\).
[in]kernelSzthe kernel size \(K\).

Note that to be mathematically correct, the kernel must be symmetric, which is why every time, the term "convolution" is mentioned is put in quotes See the implementation for details. This function is used intensively in applyFastRowBasedFilter and applyFastColumnBasedFilter.
