Detailed Description

Functor that adapts the feature scale to the local shape from the second-moment matrix (cf. [Mikolajczyk & Schmid, ECCV 2002]).

Given a feature \((\mathbf{x}, \sigma)\), the local affine shape adaptation is based on the estimation of the second moment matrix

\[ \mu(\mathbf{x}, \sigma) = (g_\sigma) * \left[ (\nabla I) (\nabla I)^T \right] (\mathbf{x}) \]

which is also the Mahalanobis distance reflecting the anisotropy of the local shape.
